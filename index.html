Maquette d'arborescence du hub d'info

Outil de prototypage pour construire et faire évoluer la structure de contenus et services du futur portail.

Document de travail pour discussion en copil.

Comment utiliser cette maquette

Cliquez sur un nœud pour voir et modifier son détail (libellé, type, TL;DR, format, URL). Utilisez « + Enfant » pour ajouter un nœud sous le nœud sélectionné, « Supprimer » pour retirer un nœud et toute sa descendance. Exportez votre travail en JSON pour le partager, en Markdown pour la rédaction de notes, ou en Mermaid pour la documentation technique.

À propos de cette maquette

Typologie de nœuds. Dix types sont prédéfinis pour qualifier chaque rubrique selon sa fonction : page éditoriale, service applicatif, simulateur, carte, renvoi externe, marketplace, kit téléchargeable, formulaire, espace authentifié, ou hub d'orientation. Cette typologie sert ensuite à arbitrer la priorisation et l'effort technique requis.

Persistance. Toutes les modifications sont conservées dans le navigateur via localStorage. Aucune donnée n'est transmise à un serveur. Pour partager votre travail, exportez un fichier JSON.

Réutilisation. Le code source est sous licence ouverte sur GitHub et peut être adapté à d'autres projets de cadrage de portails.
